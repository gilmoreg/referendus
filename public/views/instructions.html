How to use Referendus

Step 1: Create account or login

  1. Click the button in the upper right
  2. Enter a username and password
  3. To create an account, click "Sign Up" and it will create the account and sign you in automatically
  4. Or, you can Sign In to the demo account with username demo and password demo.

Step 2: Enter or edit references

To create a new reference:

  1. Click the button at the top of the page
  2. Choose Article, Book, or Website
  3. Fill in the fields and click Add. Fields marked with * are required.

To edit existing references:

  1. Click the button
  2. Change the desired fields
  3. Click Update.

To delete a reference:

  1. Click the button
  2. Answer yes or no.

Step 3: Select a view and copy to clipboard

  1. Choose a format (APA, Chicago/Turabian, or MLA) from the dropdown above.

  2. Choose a tab to display only references of that type, or All to display all references.
  3. Or, search for a tag above to see a tab with only references having that tag.
  4. The demo account has references with the tag "tag" if you want to try it out.

  5. Then click the button to copy the currently displayed references to the clipboard.
  6. You can then paste into your favorite rich text editor, such as Word, Pages, Google Docs, etc.
